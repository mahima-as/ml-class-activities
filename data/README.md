## 1. ZipDigits — Handwritten Digit Images

Used by: `Perceptron/`, `Linear Models/`, `Introduction/`

**Source:** AT&T Research Labs via the *Elements of Statistical Learning* dataset archive.

**Download:**

```bash
# Train set (~14 MB)
wget https://hastie.su.domains/ElemStatLearn/datasets/zip.train.gz
gunzip zip.train.gz
mv zip.train ZipDigits.train

# Test set (~4 MB)
wget https://hastie.su.domains/ElemStatLearn/datasets/zip.test.gz
gunzip zip.test.gz
mv zip.test ZipDigits.test
```

Or in Python:

```python
import urllib.request, gzip, shutil

def download_zipdigits(url, out_path):
    gz_path = out_path + ".gz"
    urllib.request.urlretrieve(url, gz_path)
    with gzip.open(gz_path, 'rb') as f_in, open(out_path, 'wb') as f_out:
        shutil.copyfileobj(f_in, f_out)

download_zipdigits(
    "https://hastie.su.domains/ElemStatLearn/datasets/zip.train.gz",
    "ZipDigits.train"
)
download_zipdigits(
    "https://hastie.su.domains/ElemStatLearn/datasets/zip.test.gz",
    "ZipDigits.test"
)
```

**Format:** Each line contains the digit label (0–9) followed by 256 grayscale pixel
values (16×16 image), space-separated.

**Place files at:**
- `digits/ZipDigits.train`
- `digits/ZipDigits.test`
- `Perceptron/Digits/ZipDigits.train` (and `.test`)
- `Linear Models/Digits/ZipDigits.train` (and `.test`)

> **Note:** Notebooks use hardcoded Google Drive paths
> (e.g., `/content/drive/MyDrive/...`). Update the file paths in each notebook
> to match your local setup before running.

## 3. Synthetic Dataset — 2D Perceptron Data

Used by: `Perceptron/PLA.ipynb`, `SVM/SVM.ipynb`

**Source:** Synthetically generated. Run `Perceptron/PerceptronHelper.ipynb` to
regenerate the file `Perceptron/synthetic_dataset.csv`.

To regenerate manually:

```python
import numpy as np
import pandas as pd

np.random.seed(42)
n = 50
X = np.random.randn(n, 2) * 5
y = np.where(X[:, 0] - X[:, 1] > 0, 1, -1)

df = pd.DataFrame({'x1': X[:, 0], 'x2': X[:, 1], 'y': y})
df.to_csv('Perceptron/synthetic_dataset.csv', index=False)
```

**Format:** CSV with columns `x1`, `x2`, `y` (y ∈ {-1, +1}).

## 4. Credit Approval Dataset

Used by: `Decision Trees/Decision Trees.ipynb`

**Source:** UCI Machine Learning Repository — Credit Approval dataset
(attribute names and values anonymized to protect confidentiality).

**Download:**

```bash
wget https://archive.ics.uci.edu/ml/machine-learning-databases/credit-screening/crx.data
wget https://archive.ics.uci.edu/ml/machine-learning-databases/credit-screening/crx.names
```

Or in Python:

```python
import urllib.request

base = "https://archive.ics.uci.edu/ml/machine-learning-databases/credit-screening/"
urllib.request.urlretrieve(base + "crx.data", "credit approval/crx.data")
urllib.request.urlretrieve(base + "crx.names", "credit approval/crx.names")
```

**Format:** 690 instances, 15 attributes + class label (+/-). Missing values
denoted by `?`. No header row.

**Preprocessing note:** Load with `header=None` and replace `?` with `NaN`:

```python
import pandas as pd
df = pd.read_csv("credit approval/crx.data", header=None, na_values='?')
```

**Place files at:** `credit approval/crx.data` and `credit approval/crx.names`

## 5. Play Tennis Dataset

Used by: `Decision Trees/`

**Source:** Classic teaching dataset from Quinlan (1986), "Induction of Decision Trees."

Recreate the file with this snippet:

```python
import pandas as pd

data = {
    'index':    list(range(14)),
    'outlook':  ['sunny','sunny','overcast','rainy','rainy','rainy','overcast',
                 'sunny','sunny','rainy','sunny','overcast','overcast','rainy'],
    'temp':     ['hot','hot','hot','mild','cool','cool','cool','mild','cool',
                 'mild','mild','mild','hot','mild'],
    'humidity': ['high','high','high','high','normal','normal','normal','high',
                 'normal','normal','normal','high','normal','high'],
    'wind':     ['false','true','false','false','false','true','true','false',
                 'false','false','true','true','false','true'],
    'play':     ['no','no','yes','yes','yes','no','yes','no','yes','yes',
                 'yes','yes','yes','no'],
}
pd.DataFrame(data).to_csv('tennis_decision_tree/play_tennis.csv', index=False)
```

**Place file at:** `tennis_decision_tree/play_tennis.csv`
